
# Dashboard de Análise Financeira - Power BI

Este repositório contém um relatório interativo desenvolvido no Microsoft Power BI  para análise de indicadores financeiros. O projeto visa transformar dados brutos em insights estratégicos, permitindo o acompanhamento de métricas de desempenho e saúde financeira de forma visual e intuitiva.

---

# 🚀 Tecnologias Utilizadas

Microsoft Power BI: Construção do dashboard e visualização de dados.

Power Query (M Language): Tratamento, limpeza e modelagem dos dados (ETL).

---

# 📊 Funcionalidades

Análise Temporal: Visualização da evolução financeira ao longo de períodos específicos.

Segmentação de Dados: Filtros interativos por região, produto e período para análises granulares.

KPIs Estratégicos: Indicadores de performance integrados para tomada de decisão rápida.


Modelagem de Dados: Estrutura otimizada para garantir performance e integridade das informações.
